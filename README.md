# Gym Member Activity Analysis

## Overview

This project explores gym member activity data and applies machine learning techniques to predict workout frequency based on users' physical characteristics, training habits, and fitness metrics.

The project includes data exploration, preprocessing, model training, evaluation, and comparison of multiple machine learning algorithms.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Workflow

- Data loading and exploration
- Exploratory Data Analysis (EDA)
- Data preprocessing
- Feature engineering
- Model training
- Model evaluation and comparison
- Selection of the best-performing model

## Machine Learning Models

Several classification models were trained and evaluated to predict weekly workout frequency.

The models were compared using standard evaluation metrics to identify the best-performing approach.
